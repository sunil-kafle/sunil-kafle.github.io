Daycare Data Engineering & Analytics Pipeline

End-to-end data engineering pipeline using synthetic data, SQL analytics, and operational dashboards

Overview

This project designs and implements a full data engineering pipeline for a daycare center to support operational, financial, and educational decision-making. Due to privacy constraints, fully synthetic data was generated to simulate real-world daycare operations. The solution integrates structured data modeling, data quality enforcement, ETL pipelines, analytical SQL queries, and interactive dashboards.

Problem & Business Questions

  • How many meals are required by age group and dietary preference?
  • What is parent churn and current contract status?
  • What is the total staff salary cost by month and year?
  • How much is spent on supplies by category?
  • Which suppliers offer the lowest cost for books?

Data Model & Architecture

A star schema was designed to support analytical querying while maintaining data integrity. The model includes fact and dimension tables aligned with the identified business questions.

  • Star schema driven by operational and financial use cases
  • Fact and dimension tables with enforced relationships
  • Constraints validated during ingestion

Data Engineering Approach

Synthetic Data Generation

  • Privacy-safe data generation using Faker
  • Controlled distributions to mirror real daycare operations
  • Constraint-aware synthetic datasets

Data Quality & Validation

  • Checks for NULL values, data types, and uniqueness
  • Automated logging of invalid records
  • Clean outputs isolated from raw inputs

ETL Pipeline

  • Modular extract, validate, transform, and load workflow
  • Pandas-based ingestion and transformation
  • SQLite used as the analytical warehouse

Analytics & SQL

SQL queries were written to directly answer the defined business questions using joins, aggregations, and common table expressions (CTEs). Outputs include churn analysis, cost breakdowns, and supplier comparisons.

Dashboards & Insights

  • Course Report supporting educators and parents in monitoring learning outcomes
  • Customer Report providing visibility into contracts, revenue, and location
  • Monthly Cost Report tracking staff salaries and supply expenditures

Tools & Stack

  • Python, Pandas, Faker
  • SQLite, SQL
  • Data quality validation frameworks
  • Looker Studio
  • GitHub

Collaboration
